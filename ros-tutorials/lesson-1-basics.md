# Lesson 1 - Basics

## Vocab
- ROS: Robot Operating System, the software we are learning about
- `roscore`: Command used to start ROS
- turtlesim: A ROS package that displays a turtle and allows basic interactions

## Goals
1. Start the roscore
2. Display the Turtle
